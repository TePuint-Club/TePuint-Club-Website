# 如何使用

## 安装项目所需文件
```
npm install
```

### 启动项目开发服务器（支持热重载）
```
npm run serve
```

### 编译项目到生产环境
```
npm run build
```

### 运行测试
```
npm run test
```

### 修复项目
```
npm run lint
```

### 详细设置
参照 [Configuration Reference](https://cli.vuejs.org/config/).
